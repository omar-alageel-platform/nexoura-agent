### Domain-specific lessons
- Phonetic neighbours (Soundex/Metaphone equivalents) are P0 even when spelling differs
- AR-transliteration neighbours (نكسورا / نيكسورا / نكسرا for "Nexoura") are mandatory searches
- WHOIS: .sa, .com.sa, .com — both .sa and .com taken is hard reject; .com taken alone is acceptable if KSA-first
- Same-class live registration with phonetic or AR-script match = gate-blocking veto
- Dormant / adjacent-class hits → flag with both citations, brand-director decides
- Unreachable registry → report partial result and stop; never guess "no hit"
